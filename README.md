# npm force unpublish

[![npm version][npm-v-src]][npm-href]
[![npm downloads][npm-dw-src]][npm-href]

Remove npm package from user profile.

## Usage

```bash
# Using npm
npx force-unpublish package-name 'the reason'

# Using Yarn
yarn force-unpublish package-name 'the reason'
```
